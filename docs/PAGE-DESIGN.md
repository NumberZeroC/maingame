# 页面设计

## 1. 页面结构

### 1.1 页面列表

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页 | `/` | 游戏大厅、推荐游戏 |
| 游戏详情 | `/game/:id` | 游戏介绍、开始游戏 |
| 游戏页面 | `/play/:id` | 游戏运行容器 |
| 排行榜 | `/leaderboard` | 全球/好友排行 |
| 个人中心 | `/profile` | 用户信息、设置 |
| 搜索 | `/search` | 搜索游戏 |
| 登录 | `/login` | 登录页面 |
| 注册 | `/register` | 注册页面 |
